## leinless
: clj without leiningen

<img src='dev-resources/leiningen.jpg' width=150 />

### why?
leiningen is so damn slow!

### run it
```bash
chmod 744 walk-it
./walk-it
```

### how?
Make sure to `brew install clojure`, which will come with a `clj` binary.
